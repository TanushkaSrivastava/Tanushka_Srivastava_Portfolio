## 🖤 About This Portfolio

A **pure black + green terminal-themed** developer portfolio built from scratch with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies, just clean code.

Designed to present me as a serious developer who **learns, implements, and grows** through real projects.

---

## ✨ Features

- ⚡ **Typewriter effect** — cycles through roles on the hero section
- 🖱️ **Custom cursor** — glowing green dot with a trailing ring
- 🎞️ **Smooth scroll reveals** — sections animate in as you scroll
- 🟢 **Active nav highlight** — current section glows green in the navbar
- 📱 **Fully responsive** — works on all screen sizes
- 🖥️ **Scanline texture** — subtle terminal-style overlay
- 🔄 **Spinning photo frame** — animated green ring around profile photo
- 🏷️ **Achievements section** — hackathon badges and milestones
- 📊 **Live GitHub stats** — real-time contribution graphs

## 🖥️ Running Locally

No setup needed — just 3 steps:

```bash
# 1. Clone the repository
git clone https://github.com/TanushkaSrivastava/portfolio.git

# 2. Open the folder
cd portfolio

# 3. Open index.html in your browser
# Double-click index.html  OR  use Live Server in VS Code
```

> 💡 **Tip:** Use the **Live Server** extension in VS Code for the best experience with hot reload.

## 🔧 Customisation Guide

### 📸 Add your photo
Find this in `index.html` and replace with your image:
```html
<div class="ct-photo-inner">
  <img
    src="images/your-photo.jpg"
    alt="Tanushka Srivastava"
    style="width:100%;height:100%;object-fit:cover;border-radius:50%;"
  />
</div>
```

### 🔗 Update social links
Find and replace with your real URLs in `index.html`:
```html
<a href="https://github.com/TanushkaSrivastava" ...>
<a href="https://linkedin.com/in/TanushkaSrivastava" ...>
<a href="https://twitter.com/TanushkaSrivastava" ...>
<a href="mailto:your@email.com" ...>
```

### 📊 Update GitHub stats
Replace `TanushkaSrivastava` with your exact GitHub username in `index.html`:
```html
src="https://github-readme-stats.vercel.app/api?username=TanushkaSrivastava..."
src="https://github-readme-stats.vercel.app/api/top-langs/?username=TanushkaSrivastava..."
src="https://github-readme-streak-stats.herokuapp.com/?user=TanushkaSrivastava..."
```

### ✍️ Change typewriter phrases
Open `script.js` and edit the `phrases` array:
```js
const phrases = [
  'Developer',
  'Full Stack Engineer',
  'System Design Learner',
  // Add your own here!
];
```

## 🌐 Deploy on GitHub Pages

```bash
# 1. Push your code to GitHub
git init
git add .
git commit -m "Initial portfolio commit"
git branch -M main
git remote add origin https://github.com/TanushkaSrivastava/portfolio.git
git push -u origin main
```

Then on GitHub:
> **Settings → Pages → Source → Deploy from branch → main → / (root) → Save**

Your site will be live at:
```
https://TanushkaSrivastava.github.io/portfolio
```
